# Sprite Tracker

Cloudflare Pages Fortnite sprite tracker.

## Cloudflare setup

Add these environment variables:

- EPIC_CLIENT_ID
- EPIC_CLIENT_SECRET

Deploy as a Cloudflare Pages project.

The app uses Pages Functions for OAuth handling.

## Current status

Frontend and OAuth foundation are implemented.

The final sprite inventory endpoint depends on the authorized Epic/Fortnite data source available to the application.
